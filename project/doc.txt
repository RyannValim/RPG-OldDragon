# Gerar personagens com o mesmo método

-> Estilo de Rolagem Clássico: 3d6 seis vezes distribuição entre os seis atributos.
-> Estilo de Rolagem Aventureiro: 3d6 seis vezes distribuição entre os seis atributos.
-> Estilo de Rolagem Heróico: 4d6 seis vezes (eliminando o d6 mais baixo da soma) entre os seis atributos.

----- ----- ----- SISTEMA DE JOGO ----- ----- -----
Estilo Clássico rola três dados seis vezes, cada vez que rolar o dado, o atributo correspondente à rodada do dado é a quantidade de pontos que o atributo receberá.

Estilo Aventureiro rola três dados seis vezes, cada vez que rolar o dado, o atributo correspondente à rodada do dado é a quantidade de pontos que o atributo receberá.

Estilo Heróico rola quatro dados seis vezes, porém o mestre pode decidir qual soma de pontos vai para qual atributo.
